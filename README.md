# NGO-Finder (SkillBridge)

SkillBridge is a smart web platform that connects volunteers with NGOs based on their skills.  
Users simply select their skills, apply filters, and instantly discover NGOs that match their expertise and interests.

The goal of this project is to make volunteering easier, faster, and more meaningful by bridging the gap between skilled individuals and organizations that need them.
